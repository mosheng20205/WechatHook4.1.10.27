# 初始化通讯录

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626558422](https://www.showdoc.com.cn/PCWeixinHook/11559060626558422)  
> ShowDoc page_id: 11559060626558422

##### 简要描述

- 初始化通讯录

##### 请求URL
- ` http://{{host}}:{{port}}/InitContact `
  
##### 请求方式
- POST 


##### 请求示例

```
{
    "contactSeq": 0,
    "chatRoomContactSeq": 0
}
```

##### 请求说明

|参数名|类型|说明|
|:-----  |:-----|-----                           |
|contactSeq |string   |初始值为0,下一次从返回数据中获取  |
|chatRoomContactSeq |string   |初始值为0,下一次从返回数据中获取 |

##### 返回示例

```
{
    "chatRoomContactSeq": 0,
    "contactSeq": 726677045,
    "count": 100,
    "haveNext": 1,
    "wxid": [
        "medianote",
        "floatbottle",
        "qmessage",
...内容略
    ]
}
```
