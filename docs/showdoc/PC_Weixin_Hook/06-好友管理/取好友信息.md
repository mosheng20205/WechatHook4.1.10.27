> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626558425](https://www.showdoc.com.cn/PCWeixinHook/11559060626558425)

##### 简要描述

- 用户注册接口

##### 请求URL
- ` http://{{host}}:{{port}}/GetContact `
  
##### 请求方式
- POST 

##### 请求示例

```
{
    "wxid": "medianote"
}
```
##### 请求说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|wxid |是  |string |微信ID   |
