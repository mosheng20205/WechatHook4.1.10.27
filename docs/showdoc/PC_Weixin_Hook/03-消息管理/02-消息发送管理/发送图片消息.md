# 发送图片消息

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626669771](https://www.showdoc.com.cn/PCWeixinHook/11559060626669771)  
> ShowDoc page_id: 11559060626669771

##### 简要描述

- 发送图片消息

##### 请求URL
- ` http://{{host}}:{{port}}/SendImageMsg `
  
##### 请求方式
- POST

##### 请求参数示例

```
{
    "to_wxid":"filehelper",
    "path":"C:\\1.jpg"
}
```

##### 请求参数说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|to_wxid |是  |string |发给谁   |
|path |是  |string | 图片路径 暂不支持 http 和 https 没写    |

##### 请求参数示例

```
{
    "ret": 0,
    "retmsg": "success"
}
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----    |
|ret |int   |无意义,不应该用作参考  |
|retmsg |string   |无意义,不应该用作参考  |
