# 发送文本消息

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626558418](https://www.showdoc.com.cn/PCWeixinHook/11559060626558418)  
> ShowDoc page_id: 11559060626558418

##### 简要描述

- 发送文本消息

##### 请求URL
- ` http://{{host}}:{{port}}/SendTextMsg `
  
##### 请求方式
- POST

##### 请求参数示例

```
{
    "to_wxid":"filehelper",
    "content":"myloveisverygood_haha cc加油"
}
```

##### 请求参数说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|to_wxid |是  |string |发给谁   |
|content |是  |string | 发什么内容    |

##### 请求参数示例

```
{
    "ret": 0,
    "retmsg": "success"
}
```

##### 返回参数说明 

|参数名|类型|说明|
|:-----  |:-----|-----    |
|ret |int   |无意义,不应该用作参考  |
|retmsg |string   |无意义,不应该用作参考  |
