# 发送群@文本消息_无源

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626832902](https://www.showdoc.com.cn/PCWeixinHook/11559060626832902)  
> ShowDoc page_id: 11559060626832902

##### 简要描述

- 

##### 请求URL
- ` http://{{host}}:{{port}}/SendTextMsg_NoSrc `
##### 请求方式
- POST

##### 发送示例

```
{
    "wxidorgid":"53269853218@chatroom",
    "msg":"@小聪明 @大聪明. 发消息2",
    "atlist":"<msgsource><atuserlist>thexed,wxid_ht1exrljpb6z22</atuserlist><alnode><fr>1</fr></alnode></msgsource>"
}
```

##### 发送说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|wxidorgid |是  |string |  发给哪个群 |
|msg |是  |string |  要发送的内容 注意@数量必须和atlist 中的wxid 一致！ |
|atlist |是  |string |  要@哪些人 注意是xml结构 |

##### 返回示例
```
{
    "SvrMsgID": 4888796484892654734,
    "recv_time": 1775556065,
    "recv_wxidorgid": "53269853218@chatroom"
}
```
