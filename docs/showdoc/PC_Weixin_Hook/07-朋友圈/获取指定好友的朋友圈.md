# 获取指定好友的朋友圈

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626558518](https://www.showdoc.com.cn/PCWeixinHook/11559060626558518)  
> ShowDoc page_id: 11559060626558518

##### 简要描述

- 获取指定好友的朋友圈

##### 请求URL
- ` http://{{host}}:{{port}}/mmsnsuserpage/ `
  
##### 请求方式
- POST 

##### 发送方式
```
{
    "wxid": "wxid_k......",
    "fristPageMd5": "",
    "last_snsid": "14627955735784206986"
}
```
##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|wxid |是  |string |获取谁的朋友圈   |
|fristPageMd5 |否  |string | 留空即可    |
|last_snsid |否  |string | 首页或下一页会返回,然后直接填入即可获取下一页    |

##### 返回示例

```
{
    "count": 10,
    "detail": [
        {
            "comment_reply_total": 0,
            "comment_replys": [],
            "content": "<TimelineObject><id>14627901957005521537</id><username>wxid_kqism2ws8r7s12</username>....略"
    "fristPageMd5": "",
    "last_snsid": 14397191573361930822
}
```
