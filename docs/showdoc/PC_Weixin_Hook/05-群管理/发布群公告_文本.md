##### 简要描述

- 

##### 请求URL
- ` http://{{host}}:{{port}}/setchatroomannouncement `
##### 请求方式
- POST

##### 发送示例

```
{
    "gid":"50244792721@chatroom",
    "content":"接口新公告测试",
    "url":"",
    "xml_pars":""
}
```

##### 发送说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|gid |是  |string |群ID   |
|content |是  |string |要发布的文本内容   |


##### 返回示例
```
{
    "msg": "",
    "status": 1
}
```
