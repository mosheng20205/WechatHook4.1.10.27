# 修改群聊名称

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626565253](https://www.showdoc.com.cn/PCWeixinHook/11559060626565253)  
> ShowDoc page_id: 11559060626565253

##### 简要描述

- 修改群聊名称

##### 请求URL
- ` http://{{host}}:{{port}}/oplog/edit_chatroom_name `
##### 请求方式
- POST

##### 发送示例

```
{
    "gid":"50244792721@chatroom",
    "newname":"接口修改名称"
}
```

##### 发送说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|gid |是  |string |群ID   |
|newname |是  |string |要修改的新名称   |


##### 返回示例
```
{
    "msg": "",
    "status": 1
}
```
