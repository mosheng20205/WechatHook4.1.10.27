# 取群聊信息

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626558423](https://www.showdoc.com.cn/PCWeixinHook/11559060626558423)  
> ShowDoc page_id: 11559060626558423

##### 简要描述

- 取群聊信息

##### 请求URL
- ` http://{{host}}:{{port}}/GetContact `
  
##### 请求方式
- POST


##### 请求示例

```
{
    "wxid": "21833019536@chatroom"
}
```

##### 参数

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|wxid |是  |string |群ID   |
