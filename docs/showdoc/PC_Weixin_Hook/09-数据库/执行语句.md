# 执行语句

> Source: [https://www.showdoc.com.cn/PCWeixinHook/11559060626563006](https://www.showdoc.com.cn/PCWeixinHook/11559060626563006)  
> ShowDoc page_id: 11559060626563006

-  用户表，储存用户信息

|字段|类型|空|默认|注释|
|:----    |:-------    |:--- |---|------      |
|uid    |int(10)     |否 |  |             |
|username |varchar(20) |否 |    |   用户名  |
|password |varchar(50) |否   |    |   密码    |
|name     |varchar(15) |是   |    |    昵称     |
|reg_time |int(11)     |否   | 0  |   注册时间  |

- 备注：无



##### 简要描述

- 数据库_执行语句_查询所有表

##### 请求URL
- ` http://{{host}}:{{port}}/QueryDB/execute `
##### 请求方式
- POST

##### 发送示例

```
{
    "optDbName": "contact.db",
    "SQL": "select * from sqlite_master where type='table' order by name"    
}
```

##### 发送说明

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|optDbName |是  |string |数据库名称   |
|SQL |SQL  |是 |sql语句   |
